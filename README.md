# Water Quality Prediction

A machine learning project for predicting water quality based on sensor data.

## Project Structure

- `app.py`: Main Flask application
- `sensor_interface.py`: Interface for sensor data collection and processing
- `waste_water_detector.py`: Core machine learning model for water quality prediction
- `templates/`: Contains HTML templates for the web interface

## Setup

1. Clone the repository
2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```
3. Run the application:
   ```
   python app.py
   ```
